# Guia de Contribuicao - O Livrinho

Obrigado pelo interesse em contribuir!

---

## Tipos de Contribuicao

### Sobre o Conteudo do Livro

O conteudo do livro e de autoria exclusiva de Anderson Costa Belem.

**Voce pode:**
- Fazer perguntas sobre o conteudo
- Enviar sugestoes e feedback
- Reportar erros de digitacao
- Participar de discussoes

**Como participar:** Abra uma [Issue](https://github.com/OtimizaPro/o-livrinho/issues/new) ou participe das [Discussions](https://github.com/OtimizaPro/o-livrinho/discussions)

> **Nota:** Sugestoes serao avaliadas pelo autor, mas Pull Requests com alteracoes de conteudo nao serao aceitos.

---

### Sobre o Sistema Leitor

O Sistema Leitor (plataforma de leitura online) aceita contribuicoes tecnicas.

**O que e o Sistema Leitor:**
- Interface web: [aculpaedasovelhas.org/livrinho.html](https://aculpaedasovelhas.org/livrinho.html)
- Navegacao entre capitulos
- Formatacao e acessibilidade

**Contribuicoes aceitas (com merge):**
- Melhorias na interface de leitura
- Correcao de bugs de navegacao
- Acessibilidade (leitores de tela, contraste)
- Responsividade mobile
- Scripts de build e validacao
- CI/CD e automacao

## Como Participar

### Perguntas e Sugestoes (Conteudo)

1. Va em [Issues](https://github.com/OtimizaPro/o-livrinho/issues/new)
2. Descreva sua pergunta ou sugestao
3. Use labels como `question` ou `suggestion`
4. Aguarde resposta do autor

### Contribuicoes Tecnicas (Sistema)

```bash
# 1. Fork e clone
git clone https://github.com/SEU-USUARIO/o-livrinho.git
cd o-livrinho
npm install

# 2. Crie uma branch
git checkout -b fix/descricao-do-fix

# 3. Faca alteracoes e teste
npm run lint
npm run format

# 4. Abra um Pull Request
```

## Reportar Problemas

- **Bug no sistema leitor** - Abra Issue com label `bug`
- **Erro de digitacao** - Abra Issue com label `typo`
- **Duvida sobre conteudo** - Abra Issue com label `question`
